# CLAUDE.md

This repository keeps a **single source of truth** for agent and contributor
guidance: **[AGENTS.md](./AGENTS.md)**.

All repository conventions live there — repository purpose and structure, the
`SKILL.md` format, skill pairing and spec output conventions (`specs/<topic>/`),
how to add a new skill, build/test commands, Go-skill CI and release rules,
skill design principles, testing guidelines, and the commit/PR convention.

Claude Code (and any other agent) should read and follow `AGENTS.md`.

Do not add guidance to this file. Update `AGENTS.md` instead, so the two never
diverge.
