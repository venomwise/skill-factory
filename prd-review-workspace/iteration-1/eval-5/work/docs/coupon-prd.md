# 优惠券系统 PRD

## 1. 背景

为提升用户复购率，新增优惠券功能。用户可领取优惠券并在下单时抵扣。

## 2. 功能描述

### 2.1 优惠券类型
- 满减券：满 100 减 20
- 折扣券：8 折

### 2.2 领取规则
- 用户在活动页面点击"领取"即可获得优惠券
- 每张优惠券有有效期，过期后自动失效

### 2.3 使用规则
- 下单时选择一张优惠券进行抵扣
- 优惠券金额从订单总额中扣除

## 3. 页面要求

- 活动页展示可领取的优惠券列表
- 个人中心展示"我的优惠券"
- 下单页展示可用优惠券

## 4. 验收标准

- 用户能成功领取优惠券
- 下单时能正确抵扣金额
