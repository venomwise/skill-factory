# 行业/开源最佳实践搜索参考

对于本地项目尚未实现的新功能，通过 `web_search` 获取行业标准和常见陷阱。

## 按领域的搜索主题

- **认证授权**：搜索 OAuth 2.0 标准流程、JWT 安全实践、常见漏洞
- **支付系统**：搜索支付幂等性设计、对账逻辑、退款流程
- **实时通信**：搜索 WebSocket/SSE 的成熟方案、心跳机制、断线重连
- **库存系统**：搜索超卖问题、分布式锁、最终一致性方案
- **数据同步**：搜索 CDC 模式、增量同步、冲突解决策略

## 搜索原则

- 每个主题获取 2-3 个权威来源即可，不必穷尽。
- 重点提取：常见陷阱、标准流程、安全清单。
- 如果搜索无结果或无成熟方案，在报告中标注"需自研方案，无行业标准参考"，不要强行编造。
- 将行业共识转化为对当前 PRD 的具体质疑，而非泛泛罗列标准。

**目的：** 即使本地没有参考实现，也能基于行业共识评审 PRD 的完整性和合理性。
